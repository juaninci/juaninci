## Caso de estudio 💻
<h1 align="center">la libreria</h1>
<p>este proyecto se trata de generar una catalogo para libreria con la capacidad de anadir libros, eliminar, mostrar, y filtrar segun categoria
la complejidad estaba en que fue mi primera app en vue, algo complejo fue generar el filtro de libros donde decidi generar un componente de busqueda busqueda.vue
para no usar apis aun en modo de prueba la lista de libros quedo en el archivo padre del proyecto, donde se define una constante con la lista de libros y con caracteristicas definidas
de ahi se anade una prop que recibe la info del componente podre y se procede a insertar una logica de busqueda con una constante computada que lee el texto recibido por la busqueda
y con una condicional donde si encuentra coincidencia con los datos del prop devuelve la lista que en el mismo query coinciden</p>
<p>a su vez este proyecto me hizo entender mas de raiz el funcionamiento de vue como se estructura y como se arma la app y como se comunican en si los componentes ademas de programar con el framework de vue.js</p>
